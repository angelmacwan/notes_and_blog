Python is a friendly, readable programming language used by scientists, game makers, app developers, and engineers all over the world. It was created in 1991 by a Dutch programmer named Guido van Rossum. The name comes from Monty Python's Flying Circus — a British comedy show — not the snake! Python's superpower is that it reads almost like plain English, which makes it perfect for beginners.

Why Learn Python?

  • Easy to read — code looks like sentences, not robot gibberish.
  • Huge community — millions of people use Python; help is everywhere.
  • Used everywhere — games (Pygame), websites (Django), AI/ML (TensorFlow), science, and more.
  • Free — download from python.org and start today.

Setting Up Python

Download Python from python.org/downloads and install it. Then open IDLE (it comes with Python) or any text editor. Save your files with a .py extension and run them.

You can also type python in your terminal/command prompt to open the interactive shell — type Python code and see instant results!

Your Very First Program

The tradition in programming is to print "Hello, World!" as your first program. Here's how:

print("Hello, World!")
Hello, World!

print() is a function — a named action you can trigger. Everything inside the parentheses (and inside the quotes) gets displayed on screen.

# You can print multiple lines
print("My name is Emma.")
print("I am learning Python.")
print("This is awesome!")
My name is Emma.
I am learning Python.
This is awesome!
Tip: Lines that start with # are comments. Python ignores them — they're notes for humans reading the code.

Variables — Labelled Boxes

A variable is like a labelled box where you store information. You give the box a name, then put something inside it with =.

name   = "Emma"      # a piece of text (called a string)
age    = 10           # a whole number (called an integer)
height = 1.45         # a decimal number (called a float)
is_cool = True        # True or False (called a boolean)

Python figures out the type automatically — you never have to say "this is a number" or "this is text" like in some other languages.

Naming Rules

  • Use letters, numbers, and underscores: my_score, player1
  • Can't start with a number: 1score is invalid
  • No spaces — use underscores instead: best_friend
  • Python cares about UPPER vs lower case: Name and name are different variables
# Good variable names
player_name = "Alex"
high_score  = 9999
game_over   = False

# You can update a variable
high_score = 10500   # the old value is replaced
print(high_score)    # 10500
10500

print() and input() in Depth

print() can display multiple things at once, separated by commas. input() pauses and waits for the user to type something — it always returns a string.

# print with multiple values
print("Score:", 100, "Lives:", 3)
# default separator is a space; change it:
print("2025", "06", "01", sep="-")   # 2025-06-01
# default end is a newline; change it:
print("Loading", end="")
print("...", end="\n")
Score: 100 Lives: 3
2025-06-01
Loading...
# input always returns a string
name = input("What is your name? ")
print(f"Nice to meet you, {name}!")

# Convert to a number when needed
age = int(input("How old are you? "))      # int() converts "10" -> 10
score = float(input("Enter a decimal: "))  # float() converts "9.5" -> 9.5
print(f"In 10 years you will be {age + 10}.")
Tip: Always convert input() to a number with int() or float() before doing maths. Otherwise Python will concatenate strings instead of adding numbers!

If / elif / else — Making Decisions

Python uses indentation (4 spaces or one Tab) to group lines that belong together inside an if-block.

temperature = 28

if temperature > 35:
    print("It's scorching hot!")
elif temperature > 25:
    print("Warm and sunny — great day!")
elif temperature > 15:
    print("A bit cool — bring a jacket.")
else:
    print("Cold! Stay warm.")
Warm and sunny — great day!

You can combine conditions with and / or:

age   = 13
height = 1.40

if age >= 12 and height >= 1.35:
    print("You can ride the roller coaster!")
else:
    print("Sorry, not yet.")

# or — at least one must be true
has_ticket  = False
has_pass    = True
if has_ticket or has_pass:
    print("Welcome to the park!")

For Loops — Repeat for Each Item

A for loop runs a block of code once for every item in a sequence.

# Loop over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}!")

print("Done!")

Done!

While Loops — Repeat While True

A while loop keeps running as long as a condition is True. Be careful — if the condition never becomes False, the loop runs forever!

while count <= 5:
    print(f"Count is {count}")

print("Done counting!")

A function is a named block of code you can run whenever you need it, as many times as you want. Define it with def.

Returning Values

def multiply(a, b):
    return a * b         # send the result back to the caller

answer = multiply(6, 7)
print(answer)            # 42
print(multiply(3, 3))    # 9

# Functions can call other functions
def square(n):
    return multiply(n, n)

print(square(5))   # 25

String Slicing

You can grab a section of a string using [start:stop:step]. Think of it like cutting a loaf of bread — you pick where to start and where to stop.

word = "Python"
print(word[0:3])    # Pyt   (chars 0,1,2)
print(word[2:])     # thon  (from index 2 to end)
print(word[:4])     # Pyth  (from start to index 3)
print(word[::-1])   # nohtyP (reverse the whole string!)
print(word[::2])    # Pto   (every other character)

Putting It All Together — Number Guessing Game

Now we'll build a complete game. Read through the code carefully — every part should make sense now!

import random

def play_game():
    secret = random.randint(1, 100)
    attempts = 0
    max_attempts = 7

    print("=" * 40)
    print("  GUESS THE NUMBER GAME")
    print("=" * 40)
    print(f"I've picked a number between 1 and 100.")
    print(f"You have {max_attempts} attempts. Good luck!\n")

    while attempts < max_attempts:
        guess_text = input(f"Attempt {attempts + 1}/{max_attempts} — Enter your guess: ")

        if not guess_text.isdigit():
            print("  Please enter a whole number!")
            continue

        guess = int(guess_text)
        attempts += 1

        if guess < 1 or guess > 100:
            print("  Out of range! Guess between 1 and 100.")
        elif guess < secret:
            diff = secret - guess
            if diff > 20:
                print("  Too low — way off!")
            else:
                print("  Too low — getting warmer!")
        elif guess > secret:
            diff = guess - secret
            if diff > 20:
                print("  Too high — way off!")
            else:
                print("  Too high — getting warmer!")
        else:
            print(f"\n  Correct! The number was {secret}.")
            print(f"  You got it in {attempts} attempt(s)!")
            if attempts == 1:
                print("  INCREDIBLE — first try!")
            elif attempts <= 3:
                print("  Amazing — well played!")
            return

    print(f"\n  Out of attempts! The secret was {secret}.")

play_game()
This game uses import random to load Python's built-in random module, then random.randint(1, 100) to pick a random whole number from 1 to 100.
